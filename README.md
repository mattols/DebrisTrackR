# DebrisTrackR

**Automated processing workflow to monitor supraglacial debris cover evolution**

Incorporates:

- Improved masking techniques for clouds, snow cover, and terrain shadows
- Multi-temporal image fusion


## Table of Contents

1. Tools - scripts for generating automated workflow with Landsat OLI-2, OLI, ETM+, and TM imagery
2. Visualizations - scripts for generating figures
3. Output - figures, tables, and classification rasters
4. Archive - old and unused scripts
